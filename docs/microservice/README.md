使用Go语言搭建微服务
==================
下面这些文章章节中，我们会使用Go编程语言来构建微服务， 我们一片一片的将必要集成添加进来，让它们能在Spring Cloud/Netflix OSS中的集群模式的Docker上良好运行。

如果你还不知道微服务是什么， 我们建议先读读Martin Fowler写的关于这个话题的文章。 关于更多微服务的操作模型， 我同事这篇博客文章很好的解释了关键概念。
